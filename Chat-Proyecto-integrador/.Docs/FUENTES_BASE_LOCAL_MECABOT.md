# Fuentes tecnicas usadas para ampliar la base local de MecaBot

## Referencias principales

1. NGK - lectura y diagnostico de bujias:
https://ngksparkplugs.com/en/resources/read-spark-plug

2. Michelin - guia de presion de llantas para moto:
https://www.michelin.com.au/motorbike/advice-motorbike/tyre-care/motorcycle-tyre-pressure-guide

3. Brembo - sintomas de desgaste de pastillas/balatas:
https://www.bremboparts.com/africa/en/support/maintenance/how-can-i-tell-if-my-brake-pads-are-worn-335458

4. Castrol - frecuencia de chequeo de aceite:
https://www.castrol.com/en_us/united-states/home/learn/motorcycle-maintenance/how-often-to-check-motorcycle-oil/_jcr_content.html

5. Yuasa - mantenimiento y almacenamiento de baterias:
https://www.yuasabatteries.com/resources/guides/battery-maintenance-storage/

6. NHTSA - chequeos previos de seguridad antes de cada rodada:
https://www.nhtsa.gov/road-safety/motorcycles

## Nota de uso

La base local de MecaBot se redacto en formato orientativo para usuario general.
Para compatibilidad exacta de repuestos y planes de mantenimiento, el criterio final es:

- manual OEM de la moto,
- referencia exacta de la pieza,
- y revision tecnica profesional cuando hay riesgo de seguridad.
